    What is a QR code?

    A QR (Quick Response) code is a 2D barcode that stores data such as a URL, plain text, an email address or Wi-Fi credentials. Any phone camera can read it instantly — no app required.

    The codes generated here are static: the data lives inside the code itself, so they never expire and are not tracked.

    Four ways to export

    • Image — crisp PNG with optional transparent background.
    • SVG — vector that scales infinitely for print.
    • PDF — A4 page with the QR centered and link below.
    • Card — business-card layout, ready to print.

    More than QR codes

    • Scan — read a QR code from any uploaded image.
    • Installable — add it to your phone or desktop and use it offline.
